# k6-mcps-tests-http-azdevops-pipelines
Exemplo de implementação de testes de validação com Grafana k6 para MCP Servers publicados como Web Apps (HTTP) e que inclui execução automatizada a partir de um pipeline do Azure DevOps. Inclui o build do executável do k6 com a extensão para MCPs via container.

Aplicação utilizada nos testes: **https://github.com/renatogroffe/aspnetcore10-mcp-otel-appinsights-redis_simulacaocotacoes**
